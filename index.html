Interactive relational database

Python - Flask; Java - angular.js, Bootstrap; MySQL - PyMySQL, SQLAlachemy

Collaborated with a partner to build a relational database of fictional bars, with the goal to simulate realistic transactions between each bar and their patrons. We used Python to generate 20,000+ unique transactions based on a patron's favorite bars and beers; used Python to impose patterns in the data in order to maintain realism. Developed a front-end website to visualize menus, tables, and graphs using angular.js and Bootstrap. Dropdown boxes allow users to choose what data to display.

Galaxy Simulation

Python - pygad, Pandas, Numpy; C++; GADGET

Ran simulations of various galaxies over time, mainly focusing on the effect of an Active Galactic Nuclei (a central black hole) during galactic formation. Collected data was used to track each small galaxy as they merged, forming a familial tree. This was then visualized using gmakemovie and presented to our group.

Atmospheric Chemistry and Satellites

Python; R

Our research group investigated a "clear sky" bias in atmostpheric data collected from satellites. We compared data from NASA's AQUA and TERRA satellites to IMPROVE's ground sites for the years 2010-2014. Python was used to match dates and times when the satellite passed overhead; statistical analysis in R found a significant differnce in data collected on cloudy vs. non-cloudy days.
